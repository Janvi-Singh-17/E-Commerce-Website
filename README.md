# Nike Store - Interactive E-commerce Slider

## Project Overview

This project is a modern e-commerce web application showcasing Nike products with interactive features like dynamic sliders, product previews, and a simple payment interface. Built with HTML, CSS, and JavaScript, the project demonstrates responsive design and user interaction capabilities.


## Features

- **Dynamic Product Slider:**
  - Scroll through different product categories (e.g., Air Force, Air Jordan, Blazer).
  - Updates product details dynamically when navigating through items.
  
- **Interactive Product Details:**
  - Display color and size options for products.
  - Update product image based on color selection.
  - Highlight selected sizes visually.

- **Payment Modal:**
  - Opens a payment form to collect user and card information.
  - Simulates checkout with a simple and clean design.

- **Responsive Design:**
  - Fully responsive layout optimized for desktops and mobile devices.

  
## Technologies Used

- **HTML:** Markup structure for the webpage.
- **CSS:** Styling and responsive layout using media queries.
- **JavaScript:** Interactive functionalities for sliders, modals, and dynamic updates.



## File Structure

- **`index.html`**: Main HTML file containing the structure of the webpage.
- **`style.css`**: CSS file for styling and responsive design.
- **`app.js`**: JavaScript file for dynamic functionality and user interactions.
- **`/img/`**: Folder containing images used in the application.
